# WNPlayer
WNPlayer万能播放器，使用FFmpeg最新版本(4.0.2)，没有使用FFmpeg的废弃api，支持几乎所有视频格式(avi、flv、m3u8、3gp、RTMP、RTSP、MKV、rmvb、wmv、mp4、mov)，网络和本地视频同时支持；支持自定义控制层。
